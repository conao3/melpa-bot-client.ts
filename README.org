* Overview

A TypeScript client for [[https://github.com/conao3/melpa-bot][melpa-bot]], built on AWS Lambda using the Serverless Framework. This service handles GitHub webhook events and provides a REST API for MELPA package management automation.

* Features

- Serverless architecture on AWS Lambda
- GitHub webhook integration for pull request events
- Express-based REST API with CORS support
- TypeScript with webpack bundling

* Requirements

- Node.js
- npm or yarn
- AWS CLI (configured with appropriate credentials)
- Serverless Framework

* Installation

Clone the repository and install dependencies:

#+begin_src shell
git clone https://github.com/conao3/melpa-bot-client.ts.git
cd melpa-bot-client.ts
npm install
#+end_src

* Usage

** Local Development

Run the service locally using serverless-offline:

#+begin_src shell
npx serverless offline
#+end_src

** Deployment

Deploy to AWS Lambda:

#+begin_src shell
npx serverless deploy
#+end_src

** API Endpoints

| Endpoint   | Method | Description                    |
|------------+--------+--------------------------------|
| =/=        | GET    | List available API endpoints   |
| =/hello=   | GET    | Hello world response           |
| =/ping=    | GET    | Health check                   |
| =/github=  | POST   | GitHub webhook receiver        |
